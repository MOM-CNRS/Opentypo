Une ligne du fichier correspond à une catégorie, un groupe, une série ou un type (selon les codes renseignés). Les en-têtes doivent reprendre exactement les noms du modèle.

  • Séparateur de colonnes : virgule ou point-virgule (détection automatique).
  • Listes multi-valeurs dans une cellule : séparateur || (le séparateur ## est aussi accepté pour les images).
  • Références OpenTheso : format libellé:url avec une URL http(s).
  • auteur_scientifique : {Prénom, Nom} pour chaque personne, plusieurs auteurs avec || ; si une valeur contient des virgules dans un CSV à virgules, entourer la cellule de guillemets doubles.
